Tipos de Formatação de Negrito e Itálico

Negrito

A primeira forma de usar negrito é usando a tag <b> de bold

A segunda forma de usar negrito é usando a tag <strong> de texto mais forte

Itálico

A primeira forma de usar itálico é usando a tag <i> de italic

A segunda forma de usar o itálico é usando a tag <em> de enfase

Tipos de Formatação Marcadas

Marcar

Para marcar um texto utilizamos a tag <mark> de marcação

Textos Pequenos

Para deixar textos pequenos utilizamos a tag <small>

Textos Deletados

Para textos deletados utilizamos a tag <del> de delete

Textos Inseridos

A primeira forma de utilizar textos inseridos é usando a tag a tag <u> que é mais voltada para a forma.

A segunda forma (mais recomendada) de usar textos inseridos é usando a tag <ins> de inserted
